# 🎮 ג'נתר - משחק לוח הכפל

משחק חינוכי ומהנה ללימוד לוח הכפל!

## 🎯 איך משחקים?

- הדמות זזה קדימה אוטומטית
- השתמשי בחצים או WASD כדי לשלוט בכיוון
- אספי כוכבים ⭐ ומטבעות 🪙
- פתרי תרגילי כפל כשנתקלת בקוביות סגולות 🎲
- תשובה נכונה = +30 מטבעות ✅
- תשובה שגויה = -20 מטבעות וחזרה אחורה ❌

## 📱 תאימות

המשחק עובד על:
- 💻 מחשב
- 📱 טלפון
- 📲 טאבלט

## 🚀 איך להעלות ל-GitHub Pages

1. העלי את הקובץ `index.html` ל-repository
2. לכי ל-Settings → Pages
3. בחרי Branch: main
4. המשחק יהיה זמין ב: `https://YOUR-USERNAME.github.io/genather-game`
